# 🔐 Password Strength Checker (GUI-Based)

## 📌 Overview
This project is a **GUI-based Password Strength Checker** built using Python and Tkinter.  
It evaluates the strength of a password based on security rules and provides suggestions for improvement.

---

## 🚀 Features
- 🔑 Secure password input (hidden characters)
- 📊 Strength classification:
  - Weak ❌
  - Medium ⚠️
  - Strong 💪
- 💡 Real-time suggestions for improving password strength
- 👁️ Show/Hide password functionality
- 🧹 Clear input option
- 🖥️ User-friendly GUI interface

---

## 🛠️ Tech Stack
- Python 🐍
- Tkinter (GUI)
- Regular Expressions (Regex)
