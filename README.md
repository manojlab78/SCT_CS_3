# 🔐 Password Strength Checker

## 📌 Overview
This is a GUI-based Password Strength Checker built using Python.  
It evaluates password strength and provides suggestions for improvement.

---

## 🚀 Features
- Password input with hidden text  
- Strength detection (Weak / Medium / Strong)  
- Real-time feedback  
- Suggestions for improvement  
- Show/Hide password option  
- Clear input functionality  

---

## 🛠️ Tech Stack
- Python  
- Tkinter  
- Regex  

---

## ▶️ How to Run

```bash
python app.py
